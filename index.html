PRINCIPALES ESTILOS CSS

ESTILOS DESCRIPCION
Colors Define el color del texto. Ejm: {color: white;}
Backgrounds Permite definir simultáneamente todas las propiedades relacionadas con el fondo de cualquier elemento. Ejm: {background-color: #005883;}
Borders Definir los bordes de un bloque. Ejm: {border-width: 1px;}
Margins Establece el margen para los cuatro lados. Ejm: {margin: 20px 0px 0 -20px;}
Text Este estilo lo utilizaremos para darle un aspecto diferente al párrafo
Fonts Podemos cambiar el tamaño del texto, el color del mismo, su anchura, su estilo e incluso su tipo de letra. Ejm: {font-style: italic;}
Height Especifica la altura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento. Ejm: {height: 25px;}
Width Especifica la anchura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento. Ejm: {width: 25em border-box;}
Links Lineas de texto que nos redireccionan a otra ventana o pagina. Ejm: a:link {color: #265301;}
Lists La propiedad de estilo de lista permite definir de todos los parámetros. Ejm: {list-style: circle;{
Tables La propiedad de estilo de tablas permite definir de todos los parámetros. Ejm: table {width: 100%;border: 1px solid #000;}
Position La propiedad position especifica cómo un elemento es posicionado en el documento. Ejm: {position: relative;}
Overflow La propiedad overflow especifica: si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque. Ejm: {overflow: visible;}
Float El estilo “float” sirve para alinear un elemento, haciendo que el texto de alrededor del mismo se agrupe entorno a dicho elemento. Ejm: {float:right;}
